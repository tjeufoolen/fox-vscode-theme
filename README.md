# Fox VSCode Theme

A simple fox orange vscode color theme

## Getting Started

1. Clone this project
2. Make changes in `themes/Fox-color-theme.json`
3. Use `vsce package`
4. Use `vsce publish`
5. The changes have been pushed and will be visible in ~5minutes üòç
